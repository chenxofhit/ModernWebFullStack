CSS 定位（Position）示例

1. 静态定位（Static）- 默认

静态

position: static 是默认值，元素按照正常文档流进行布局。

top、right、bottom、left 和 z-index 属性不会影响静态定位的元素。

2. 相对定位（Relative）

静态
相对
静态

position: relative 使元素相对于其正常位置进行定位。

上面的第二个盒子使用了 top: 20px; left: 20px;，所以它向下和向右移动了 20px。

相对定位不会影响其他元素的位置，也不会使元素脱离文档流。

3. 绝对定位（Absolute）

静态
绝对
静态

position: absolute 使元素相对于最近的已定位祖先元素进行定位。

如果没有已定位的祖先元素，则相对于文档的 body 元素定位。

上面的第二个盒子使用了 top: 40px; right: 40px;，所以它位于容器右上角。

绝对定位会使元素脱离文档流，不占据空间，其他元素会忽略它的存在。

4. 固定定位（Fixed）

静态
固定

position: fixed 使元素相对于浏览器窗口进行定位。

固定定位的元素不会随页面滚动而移动。

上面的紫色盒子使用了 bottom: 20px; right: 20px;，所以它固定在浏览器窗口的右下角。

固定定位会使元素脱离文档流，不占据空间。

向下滚动页面，观察紫色盒子的位置。

5. 粘性定位（Sticky）

粘性

向下滚动页面，观察上面的红色盒子如何"粘"在页面顶部。

这是一些额外的内容，用于演示粘性定位的效果。向下滚动页面，观察红色盒子的行为。

position: sticky 是相对定位和固定定位的混合。

元素在滚动到特定阈值（如 top: 0）之前表现为相对定位，之后表现为固定定位。

粘性定位常用于导航栏或标题，使其在滚动时保持可见。

定位方式总结

定位方式 特点 常见用途
静态 (static) 默认值，按正常文档流布局 普通内容布局
相对 (relative) 相对于正常位置偏移，不脱离文档流 微调元素位置，为绝对定位子元素创建参考
绝对 (absolute) 相对于最近的已定位祖先元素，脱离文档流 弹窗、工具提示、徽章、自定义位置的元素
固定 (fixed) 相对于视口定位，不随页面滚动，脱离文档流 固定导航栏、返回顶部按钮、固定广告
粘性 (sticky) 根据滚动位置在相对和固定之间切换 滚动时固定的标题、导航栏
