Grid 网格布局教学

Grid 是二维布局系统，可以同时控制行和列。适合整页布局、复杂网格、响应式设计。

1. 基本三列等宽

1fr 表示「1 份」剩余空间，三列 1fr 1fr 1fr 即三等分。

.grid-basic { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 15px; }

2
3

2. repeat() 简化写法

repeat(3, 1fr) 等价于 1fr 1fr 1fr。

grid-template-columns: repeat(3, 1fr);

3. 不等宽列

1fr 2fr 1fr 表示中间列是两侧的 2 倍宽。

grid-template-columns: 1fr 2fr 1fr;

4. 固定宽度 + 弹性

侧边栏固定 200px，主区域自适应。

grid-template-columns: 200px 1fr 100px;
200px
1fr 自适应

5. 整页布局（grid-template-areas）

用命名区域定义布局，直观易读。

grid-template-areas: "header header" "sidebar main" "footer footer";
Header 顶部导航
Sidebar 侧边栏
Main 主内容区域
Footer 页脚

Flex vs Grid 选择建议

  • 一维排列（横排或竖排）→ 用 Flex
  • 二维网格（行+列同时控制）→ 用 Grid
  • 整页布局（header/sidebar/main/footer）→ 用 Grid
