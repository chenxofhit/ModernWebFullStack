Flex布局教学

Flexbox（弹性盒子）是CSS3中的一种布局模式，它可以让我们更加方便、灵活地设计响应式布局。本教程将通过实例讲解Flex布局的基本概念和常用属性。

1. 基本的Flex容器

要创建一个Flex布局，首先需要将一个元素设置为Flex容器。这通过设置display: flex来实现。

.flex-container-basic { display: flex; }

2. Flex方向 (flex-direction)

Flex容器有一个主轴和一个交叉轴。默认情况下，主轴是水平的，从左到右。我们可以使用flex-direction属性来改变主轴的方向。

row（默认值）：从左到右

.flex-direction-row { display: flex; flex-direction: row; }

3. 主轴对齐方式 (justify-content)

justify-content属性定义了项目在主轴上的对齐方式。

.justify-content-demo { display: flex; justify-content: flex-start; }

  • flex-start：（默认值）项目靠主轴起点对齐
  • flex-end：项目靠主轴终点对齐
  • center：项目在主轴上居中对齐
  • space-between：项目均匀分布，第一个项目靠起点，最后一个项目靠终点
  • space-around：项目均匀分布，每个项目两侧的间隔相等
  • space-evenly：项目均匀分布，项目之间的间隔相等，包括与容器边缘的间隔

5. Flex换行 (flex-wrap)

默认情况下，项目都排在一条线上。flex-wrap属性定义了如果一条轴线排不下，如何换行。

.flex-wrap-demo { display: flex; flex-wrap: nowrap; }

6. Flex项目属性

Flex布局不仅为容器提供了属性，也为项目（子元素）提供了属性。

flex-grow（扩展比例）

flex-grow定义项目的放大比例。默认为0，即如果存在剩余空间，也不放大。

.flex-grow-1 { flex-grow: 1; } .flex-grow-2 { flex-grow: 2; }

flex-grow: 1

flex-shrink（收缩比例）

flex-shrink定义了项目的缩小比例。默认为1，即如果空间不足，该项目将缩小。

.flex-shrink-1 { flex-shrink: 1; width: 200px; } .flex-shrink-3 { flex-shrink: 3; width: 200px; }

flex-basis（基础尺寸）

flex-basis定义了在分配多余空间之前，项目占据的主轴空间。默认值为auto，即项目的本来大小。

.flex-basis-auto { flex-basis: auto; } .flex-basis-100 { flex-basis: 100px; } .flex-basis-200 { flex-basis: 200px; }
auto

flex（简写属性）

flex是flex-grow, flex-shrink和flex-basis的简写。默认值为0 1 auto。

.flex-1 { flex: 1; } /* 相当于 flex: 1 1 0%; */ .flex-auto { flex: auto; } /* 相当于 flex: 1 1 auto; */ .flex-none { flex: none; } /* 相当于 flex: 0 0 auto; */

7. 对齐单个项目 (align-self)

align-self属性允许单个项目有与其他项目不一样的对齐方式。它可以覆盖align-items属性。

#special-item { align-self: auto; }
