📚 GET 和 POST 的主要区别：

  • 数据位置：GET方法将数据放在URL中（查询字符串），POST方法将数据放在请求体中
  • 可见性：GET的数据在URL中可见，POST的数据在请求体中不可见
  • 安全性：POST相对更安全，适合传输敏感信息（如密码）
  • 数据长度：GET有URL长度限制，POST可以传输更多数据
  • 用途：GET用于获取数据，POST用于提交/修改数据
  • 缓存：GET请求可以被浏览器缓存，POST通常不会被缓存
  • 书签：GET请求可以保存为书签，POST不可以
